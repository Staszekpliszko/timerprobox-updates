# TimerProBOX Updates

Publiczny kanał aktualizacji aplikacji TimerProBOX (OTA).

- `version.json` — manifest najnowszej wersji (pobierany przez aplikację)
- Releases — paczki `update-X.Y.Z.tar.gz` (podpisane Ed25519)

Kod źródłowy aplikacji jest w prywatnym repozytorium. Tu trafiają wyłącznie
zbudowane, podpisane paczki. Aplikacja instaluje paczkę tylko po weryfikacji podpisu.
